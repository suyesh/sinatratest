  • If You Need Responsive Content Then Use bootstrap Grid Inside tabs-content

Color Themes

Light
Light Is A Default Theme You Dont Need To Apply Any Extra Class
Dark
Class For Dark Theme Is "dark" ,If You Need The Dark Textboxes and inputs You Can Apply "dark-input"
Grey
Class For Grey Theme Is "gery" ,If You Need The grey Textboxes and inputs You Can Apply "grey-input"
  • * Tabs-Content Defaults To Inherit Color From Parent
  • If You Are Using dark or grey theme then apply appropriate color to Tab-pane [".tab-pane{color:white}"]

Animations

Directional Fade
address "tabs-animated" class on div.tabbable
  • "tabs-animated" Class Automatically Adds Directional Animations To Tabs According To Tabs Position

Responsiveness

Hide Label Below 980px Widht
Apply class "hide-label-980"
Hide Label Below 768px Widht
Apply class "hide-label-768"
Hide Label Below 480px Widht
Apply class "hide-label-480"
  • Responsive Also Requires Little Change In Markup Of tabs Title [Shown Below]
  • Wrap Title With span tag and place icons either before or after span tag inside a tag
  • Use This Class Only If You USe Icons In Tabs

Css3 Effects

Papper Stack

  • Add Class "papper-stack"
  • Default Direction Of Papper Stack Is Bottom-Right
  • More Classes For Papper Stack As Below
"papper-stack-auto"
Change Direction According To Tabs Direction(LEFT,RIGHT,....)
"papper-stack-BR"
Effect On Bottom-Right (Default)
"papper-stack-BL"
Effect On Bottom-Left
"papper-stack-LT"
Effect On Left-Top
"papper-stack-TR"
Effect On Top Right
*
"papper-stack" Is Not Required When Using Above Css Class

Shadow

"shadow"
This Class Adds Shadow On Tabs-Title And Tabs-Content
"shadow-box-only"
Only Add Shadow On Tabs-Content

Track Url

  • This Method Requires Extra Javascript File Shown In File Setup (tabs-addon.js)
  • Apply "track-url" class
  • If You Want To Auto Scroll According To Tabs Position Then Apply "auto-scroll"
  • URL Example: http://www.yoursite.com#panel5

List Of All Css Classes Can Be Added To div.tabbable

Class Name Description
custom-tabs All Styles Come From This Class (Must Be Required)
flat Removes Border Radius From Tabs-Title And Tabs-Content
flat-all Removes Border Radius Of input,select,.btn & textarea
shadow Adds css3 shadow to tabs-title and tabs-content box
shadow-box-only Only Adds Shadow To tabs-content box
tabs-left Align Tabs To Left Side
tabs-right Align Tabs To Right Side
tabs-below Align Tabs To Bottom (Refer Tab Position For More Info)
dark Apply Dark Theme To Tabs Control (inherits font-color from parent)
dark-input Apply Dark Theme To Inputs (Only Works With dark Class Applied)
grey Apply Grey Theme To Tabs Control (inherits font-color from parent)
dark-input Apply Grey Theme To Inputs (Only Works With grey Class Applied)
hide-label-980 Hide Label Below 980px Width (refer Responsiveness)
hide-label-768 Hide Label Below 768px Width (refer Responsiveness)
hide-label-480 Hide Label Below 480px Width (refer Responsiveness)
tabs-animated Apply Directional Fade And Slide Effect According To Tab Alignment
mini Reduces A Padding Of Tabs-Title  (8px 10px)
small Reduces A Padding Of Tabs-Title  (10px 15px)
large Reduces A Padding Of Tabs-Title  (15px 20px)
Default Padding For Tabs-Title Is (20px 25px)
papper-stack Add Papper Stack Effect (Bottom-Right)
papper-stack-auto Add Papper Stack Effect Direction Changes According To Tab Alignment
papper-stack-BR Add Papper Stack Effect (Bottom-Right)
papper-stack-BL Add Papper Stack Effect (Bottom-Left)
papper-stack-LT Add Papper Stack Effect (Left-Top)
papper-stack-TR Add Papper Stack Effect (Top-Right)
track-url Track Url For Cross Site Or Cross Page Targetting
auto-scroll Works With track-url Auto Scroll The Page To The Tab Control
